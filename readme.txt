== Description ==
Wordpress plugin to recieve donation in multiple currencies in one click. Can be used in the sidebar, posts and pages.
WordPress PayPal Donations Plugin uses IPN to ensure the values are correct and you can also chose to display your total donations to date.

For screenshots, more information and updates, please visit:
http://www.shrawasti.com/about/development-center/

Usage:
There are three ways you can use this plugin:

   1. Add the trigger text <!--WP_PAYPAL_DONATION_--> to a post or page
   2. Call the function from a template file: <?php echo paypalDonationReceive(); ?>
   3. Use the WP Paypal Donation Widget from the Widgets menu.

== Installation ==

1. Unzip and Upload the folder 'wp-paypal-donation' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings and configure the options eg. your email, Subject text etc.
4. Use the Widget or add the trigger text <!--WP_PAYPAL_DONATION_--> to a post or page where you want the donate button to appear.
